## ✨ Features
- ✔️ Real-Time **WPM, Accuracy & Errors**  
- ✔️ Random **Typing Test Passages**  
- ✔️ Elegant **Royal Certificates** with **barcode**  
- ✔️ Offline/Online **status detection**  
- ✔️ Responsive for **desktop & mobile**  
- ✔️ Anti-cheat → Paste disabled 🚫
